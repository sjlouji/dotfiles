# dotfiles

Personal macOS dotfiles — synced across multiple Macs via GitHub.

## Fresh Mac setup

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/sjlouji/dotfiles/main/bootstrap.sh)"
```

That's it. The bootstrap script handles everything:

- Detects chip (Apple Silicon vs Intel) and sets paths accordingly
- Installs Xcode CLI tools, Homebrew, Oh My Zsh
- Clones this repo, installs all packages via Brewfile
- Creates all symlinks
- Sets up SSH key, VS Code extensions, macOS defaults
- Registers the auto-sync LaunchAgent

## Daily sync

| Command | What it does |
|---|---|
| `dsync` | Commit local changes + push to GitHub |
| `dsync pull` | Pull from GitHub + re-apply symlinks |
| `dsync status` | Show local vs remote diff |

The auto-sync agent also checks every 30 minutes and notifies you when there are new commits to pull.

## Structure

```
dotfiles/
├── bootstrap.sh          # One-liner installer
├── Brewfile              # All packages
├── zsh/                  # Shell config
├── git/                  # Git config + global gitignore
├── ssh/                  # SSH config (no keys)
├── claude/               # Claude AI config + CLAUDE.md
├── vscode/               # VS Code settings, keybindings, extensions
├── macos/                # macOS defaults + LaunchAgent
├── scripts/              # symlink.sh, ssh-keygen.sh
├── sync/                 # dsync, auto-sync, git hooks
├── bin/                  # Custom CLI scripts (on PATH)
└── .local/               # Machine-specific (gitignored)
```

## Custom scripts (`bin/`)

All scripts are on PATH automatically. Run any with `--help` for usage.

**System**
- `flushdns` — flush DNS cache
- `killport <port>` — kill whatever is on a port
- `clearcache [--dry-run]` — clear dev + macOS caches
- `diskcheck` — disk usage summary
- `sysinfo` — system overview

**Files**
- `cppath [file]` — copy absolute path to clipboard
- `mkcd <dir>` — mkdir + cd
- `batchrename '<pattern>'` — bulk rename with sed
- `imgresize <img> <px>` — resize image (no deps)
- `convert2jpg [quality]` — batch convert to JPG
- `zipdir <dir>` — zip with timestamp
- `trash <files>` — safe rm (moves to Trash)

**Git**
- `gnuke [--dry-run]` — delete merged branches
- `gpr` — interactive PR checkout (gh + fzf)
- `grepo <org> [dir]` — clone all org repos
- `ghealthcheck` — repo health summary
- `gclean [--dry-run]` — remove .DS_Store, prune, GC

## Machine-specific config

Everything in `.local/` is gitignored and machine-specific:

```
.local/
├── machine.sh          # Auto-generated by bootstrap (role, paths, identity)
├── .zshrc.local        # Optional shell overrides
├── Brewfile.local      # Machine-only brew packages
├── .gitconfig-work     # Work git identity
└── .gitconfig-personal # Personal git identity
```

## Security

- The `pre-push` git hook blocks accidental commits of secrets or `.local/` files
- SSH private keys are never stored here
- Secrets should use `op read` (1Password CLI) not hardcoded values

## Updating

After editing any dotfile:
```sh
dsync        # commit + push
```

On the other Mac:
```sh
dsync pull   # pull + re-symlink
reload       # reload shell
```
